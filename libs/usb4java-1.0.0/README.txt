The lib directory contains the following JAR files:

usb4java-*.jar           (The full JAR with native libraries included)
usb4java-*-nonatives.jar (The same without the native libraries)
usb4java-*-natives.jar   (Only the native libraries)

commons-lang3-*.jar      (Apache Commons Lang library needed by usb4java)
usb-*.jar                (javax.usb library needed by usb4java)

usb4java-*-sources.jar   (The source code of usb4jaba)
usb4java-*-javadoc.jar   (The JavaDoc of usb4java)
